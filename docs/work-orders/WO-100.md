## 작업 내용

### 1. 앱 버전 반환 유틸 함수 생성

`APP_CONFIG`에서 버전 정보를 읽어 포맷된 문자열을 반환하는 함수를 만든다.

```typescript
// File: src/shared/lib/getAppVersion.ts
import { APP_CONFIG } from '@shared/config/constants';

/** "몽글날씨 v1.0.0" 형태의 문자열을 반환 */
export function getAppVersion(): string {
  return `${APP_CONFIG.NAME} v${APP_CONFIG.VERSION}`;
}
```

### 2. 단위 테스트

```typescript
// File: src/shared/lib/__tests__/getAppVersion.test.ts
import { getAppVersion } from '../getAppVersion';

describe('getAppVersion', () => {
  it('앱 이름과 버전을 포함한 문자열을 반환한다', () => {
    const result = getAppVersion();
    expect(result).toContain('몽글날씨');
    expect(result).toMatch(/v\d+\.\d+\.\d+/);
  });
});
```
