# 03 — 애니메이션 & 전환 효과

> 앱 전반의 애니메이션 스펙과 전환 효과를 정의합니다.

---

## 기본 프리셋

### Timing (시간 기반)
```typescript
// shared/lib/animations.ts에서 import
export const TIMING = {
  fast: { duration: 150 },
  normal: { duration: 300 },
```

### Spring (물리 기반)
```typescript
export const SPRING = {
  default: { damping: 15, stiffness: 150 },

```

---

## 공통 애니메이션 훅

| 훅 | 용도 |
|----|------|
| `useFadeIn(delayMs?)` | 페이드 인 등장 |
| `useSlideInRight(delayMs?)` | 오른쪽에서 슬라이드 |
| `usePopIn(delayMs?)` | 팝 등장 (scale 0→1) |
| `usePressScale(scaleDown?)` | 버튼 누를 때 축소 |
| `usePulse(targetScale?, durationMs?)` | 반복 펄스 |

---

## ⚠️ 주의사항

- `withRepeat(-1)` + Reduced Motion = **ANR 크래시**
- 반드시 `useReducedMotion()` 체크 후 사용

```typescript
const reduceMotion = useReducedMotion();
if (reduceMotion) return; // 반복 애니메이션 건너뜀
```
