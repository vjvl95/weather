# 01 — 공통 토큰 구조

> 앱 전반에서 사용하는 디자인 토큰 구조를 정의합니다.

---

## 토큰 계층

```
ThemeColorTokens
├── background     # 화면 배경
├── surface        # 카드/컨테이너 배경
├── primary        # 주요 액션 색상
├── secondary      # 보조 액션 색상
├── accent         # 강조 색상
├── text
│   ├── primary    # 본문 텍스트
│   ├── secondary  # 보조 텍스트
│   └── disabled   # 비활성 텍스트
├── border         # 테두리
├── error          # 에러 상태
├── success        # 성공 상태
└── ...
```

## 타입 정의 위치

```
src/features/theme-manager/model/types.ts
```

## 사용 방법

```typescript
import { useTheme } from '@features/theme-manager';

function MyComponent() {
  const { colors } = useTheme();
  return (
    <View style={{ backgroundColor: colors.background }}>
      <Text style={{ color: colors.text.primary }}>...</Text>
    </View>
  );
}
```
