# 06 — 미디어 에셋 관리

> 앱에서 사용하는 미디어 파일(이미지, 사운드, 비디오 등)의 관리 규칙을 정의합니다.

---

## 에셋 폴더 구조

```
assets/
├── images/
│   ├── icons/        # 앱 아이콘 관련
│   └── ...
├── fonts/            # 커스텀 폰트 (.ttf, .otf)
└── sounds/           # 사운드 파일 (.mp3)
```

---

## 이미지 에셋

| 항목 | 권장 사양 |
|------|-----------|
| 프로필/썸네일 | 정사각형 300×300, JPEG, ~50KB |
| 아이콘 | SVG 또는 PNG 투명 배경 |
| 스플래시 | 2048×2048, PNG |

---

## 폰트 에셋

```typescript
// app/_layout.tsx에서 로딩
import { useFonts } from 'expo-font';

const [fontsLoaded] = useFonts({
  'FontName-Bold': require('../assets/fonts/FontName-Bold.ttf'),
  'FontName-Regular': require('../assets/fonts/FontName-Regular.ttf'),
});
```

---

## 사운드 에셋 (expo-av 사용 시)

```typescript
// 정적 require 모듈 맵 사용 (동적 require 불가)
const SOUND_MAP = {
  success: require('../assets/sounds/success.mp3'),
  error: require('../assets/sounds/error.mp3'),
} as const;
```
